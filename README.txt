瀚海星云的油猴(greasemonkey)的脚本，添加了一些快捷键

Chrome用户和Firefox用户可以直接点击 这里安装

Firefox需要安装greasemonkey, 使用Firefox 4.0beta的用户可以安装nightly build的greasemonkey, 见 https://arantius.com/misc/gm-nightly/

感谢BetterNewsmth的启发

使用

版面

j/k 上下选择帖子，可自动翻页
n/p 翻页
l或者回车 同主题打开 L 仅打开当前的主题
t切换同主题模式
r刷新
w发帖
s显示版面搜索
h显示帮助

帖子

j/k 上下选择回复，可自动翻页
n/p 翻页
r回复当前主题 R回复楼主
q返回上一页
s显示版面搜索
t切换引文显示
h显示帮助
